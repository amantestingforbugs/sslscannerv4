Subfinder Integration
Continuous subdomain discovery powered by ProjectDiscovery Subfinder
🔍

How it works: Subfinder discovers new subdomains from your project's root domains. New subdomains are stored, deduplicated, and automatically SSL-scanned.

Runs every 30 minutes (configurable per project). Only new subdomains trigger SSL scans.

⚠ Subfinder binary not found

Install subfinder on your server for full functionality. Without it, the integration will log warnings but won't crash.

# Install subfinder (Linux/ARM)
go install -v github.com/projectdiscovery/subfinder/v2/cmd/subfinder@latest

# Or download binary from:
# https://github.com/projectdiscovery/subfinder/releases
